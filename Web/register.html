ข้อมูลผู้มีอำนาจลงนามนิติบุคคล / กิจการ
เอกสารที่สมัครใช้บริการ:
1. หนังสือแสดงความตกลงในการขอรับ Username และ Password เพื่อขอใช้ระบบขอเอกสารทาง อิเล็กทรอนิกส์ (E-Payment)
1.1 ผู้มีอำนาจลงนามนิติบุคคลตามหนังสือรับรองลงนาม และลงวันที่
1.2 ตราประทับ (ถ้ามี)
1.3 ผู้รับมอบอำนาจลงลายมือชื่อ และลงวันที่ (ถ้ามี)
2. หนังสือรับรองบริษัท
3. สำเนาบัตรประชาชนผู้มีอำนาจลงนามแทนนิติบุคคล พร้อมเซ็นต์สำเนาถูกต้อง
4. หนังสือมอบอำนาจ พร้อมสำเนาบัตรประจำตัวประชาชนของผู้มอบอำนาจ (กรณีมอบอำนาจ)

สถานที่ติดต่อ:
ติดต่อ ยื่นเอกสารฝ่ายการเงินและควบคุมงบประมาณ องค์การกระจายเสียงและแพร่ภาพสาธารณะแห่งประเทศไทย (Thai PBS)

การรับ Username และ Password :
ฝ่ายการเงินและควบคุมงบประมาณ ตรวจสอบเอกสารและยืนยันการลงทะเบียน แจ้งยืนยันผลการลงทะเบียน เพื่อใช้งานระบบออกใบกำกับภาษีออนไลน์กับทาง ส.ส.ท. ส่งทาง E-mail (สามารถใช้บริการระบบได้)
